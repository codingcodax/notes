The personal pronouns replace the [nouns](/learn/english/noun) that function as the subject in a sentence.
- number
| singular | plural |
| -------- | ------ |
| I        | We     |
| you      | you    |
| he       | they   |
| she      |        |
| it       |        |

- genre
| masculine | feminine | neutral |
| --------- | -------- | ------- |
| he        | she      | it      |

- person
| first | second | third |
| ----- | ------ | ----- |
| I     | you    | he    |
|       |        | she   |
|       |        | it    |
