# Strict Mode

> Reference: [Strict Mode](https://react.dev/reference/react/StrictMode)

Strict mode is a way to enable additional checks and warnings in React.

```tsx
import { StrictMode } from "react";

const App = () => {
  return (
    <StrictMode>
      <h1>Hello, world!</h1>
    </StrictMode>
  );
};
```
